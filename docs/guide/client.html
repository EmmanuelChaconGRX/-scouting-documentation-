# 02 Cliente

El lado del cliente del proyecto Scouting fué desarrollado con el framework Vue.js (opens new window)

# Configuración

  1. Dentro del directorio scouting-client ejecutamos el comando npm install para la instalacion de dependencias necesarias.
npm install
  1. Una vez instalada las dependencias ejecutamos el comando npm run serve para levantar el servidor.
npm run serve
  1. Una vez iniciado el servidor ingresamos a la url http://localhost:8080/
